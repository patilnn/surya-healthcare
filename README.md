# Surya Healthcare

A modern React hospital website built with Vite and TailwindCSS. Features responsive design with patient-friendly navigation, service showcase, and contact functionality. Perfect for healthcare providers seeking a professional online presence with clean, modern styling.

## Features

- **Responsive Design**: Mobile-first approach with TailwindCSS
- **Modern Stack**: React 19, Vite, React Router DOM
- **Clean Architecture**: Component-based structure
- **Professional UI**: Healthcare-focused design
- **Fast Performance**: Optimized build with Vite

## Pages

- **Home**: Hero section, services overview, doctor introduction
- **About**: Hospital information and mission
- **Services**: Medical specialties and treatments
- **Contact**: Patient inquiry and appointment booking

## Technologies Used

- **React 19**: Modern React with latest features
- **Vite**: Fast build tool and development server
- **TailwindCSS**: Utility-first CSS framework
- **React Router DOM**: Client-side routing
- **Lucide React**: Beautiful icons

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/patilnn/surya-healthcare.git
cd surya-healthcare
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
